# 🏡 Association Mini Maxi - Site web

Site officiel de l’association **Mini Maxi de Bretteville-du-Grand-Caux**.

Ce projet est un site web simple réalisé en **HTML, CSS et JavaScript**, et hébergé gratuitement via **GitHub Pages**.

---

## 🌐 Démo en ligne

👉 https://Aghilas01.github.io/mini-maxi-site/


---

## 📁 Structure du projet


mini-maxi-site/
│
├── index.html # Page principale du site
├── style.css # Design du site
└── script.js # Fonctionnalité du carrousel


---

## 🎯 Objectif du site

Ce site a pour but de :
- Présenter l’association Mini Maxi
- Informer sur les activités proposées
- Partager les événements
- Publier des photos des activités
- Permettre de contacter l’association

---

## 🧱 Technologies utilisées

- HTML5
- CSS3
- JavaScript (vanilla)
- GitHub Pages 

---

## ✨ Fonctionnalités

- ✔ Site responsive (ordinateur + mobile)
- ✔ Menu de navigation simple
- ✔ Section accueil avec présentation
- ✔ Liste des activités et événements
- ✔ Galerie photos en grille
- ✔ Carrousel d’images interactif
- ✔ Page contact
